# Closet Shopper MVP

A gorgeous, interactive, mobile-first Web App MVP (iOS feel) for 'Closet Shopper'. High-end minimalist fashion aesthetic.

## Feature Flow
1. **Wardrobe Scanner**: User uploads/uses mock photo. Simulated AI scanning/analysis extracts style, color, and attributes.
2. **Closet/Wardrobe**: Scanned items categorized by type (Shirts, Pants, Shoes, etc.).
3. **Style Profile**: Generated based on scanned items and user preferences.
4. **Outfit Builder / Matcher**: Select an item to automatically generate/match pants and shoes from mock Shopify Spring '26 catalogs.
5. **Shopify Shop**: Personalized recommendations with "Buy via Shop" buttons.
6. **Checkout Path**: Mock Shopify Shop sign-in and checkout flow.
